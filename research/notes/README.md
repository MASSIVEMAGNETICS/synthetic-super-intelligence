# Research Notes

This directory is for your personal research notes, observations, and ideas.

## Organization

You can organize notes in any way that works for you:

- By topic
- By date
- By project
- By author

## Markdown Format

We recommend using Markdown format for easy reading and version control:

```markdown
# Research Note: [Topic]

Date: YYYY-MM-DD
Related Papers: [paper1, paper2]

## Summary

Your summary here...

## Key Points

- Point 1
- Point 2

## Questions

- Question 1
- Question 2

## Next Steps

- [ ] Task 1
- [ ] Task 2
```

## Linking to Papers

You can reference papers using their filenames or titles to create connections between your notes and the research papers in the system.
